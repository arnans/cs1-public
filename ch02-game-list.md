# Chapter 2 — Game List for Game Controller Activities

## One-Key Games (Activity 2.4)

These games require only a single key (e.g., Space bar) to play. Students use a vibration sensor or button as a one-key game controller.

| Name | Original Link | Short Link | Description |
|------|--------------|------------|-------------|
| Chrome Dino | https://chromedino.com/ | d.gogoboard.org/dino | The classic Chrome offline dinosaur game. Press Space to jump over obstacles. |
| Flappy Bird | https://thorium.rocks/games/flappy-bird/ | d.gogoboard.org/flappy | Classic fun. Tap to flap and navigate through pipes. |
| Geometry Dash | https://www.crazygames.com/game/geometry-dash-online | d.gogoboard.org/geodash | Similar to Chrome Dino but with more tricks and rhythm-based obstacles. |
| Canabalt | https://canabalt.com/ | d.gogoboard.org/canabalt | Fast-paced endless runner. More challenging — jump between rooftops at high speed. |

## Multi-Key Games (Activities 2.5 and 2.6)

These games require more than one key (e.g., arrow keys). Students use the GoGo Touch directional pads or multiple sensors to build a multi-key controller.

| Name | Original Link | Short Link | Description |
|------|--------------|------------|-------------|
| Pac-Man | https://elgoog.im/pacman/ | d.gogoboard.org/pacman | The classic maze game. Use 4 arrow keys to eat pellets and avoid ghosts. |
| Timberman | https://www.miniplay.com/game/timber-man | d.gogoboard.org/timber | Classic fun. Chop the tree by pressing left or right to avoid branches. 2 keys. |
| Snake | https://www.crazygames.com/game/snake-io | d.gogoboard.org/snake | Easy and fun. Guide the snake to eat food and grow without hitting walls. 4 keys. |
| Tetris | https://play.tetris.com/ | d.gogoboard.org/tetris | Classic puzzle game. Rotate and move falling blocks to complete rows. 4 keys. |
| Space Invaders | https://freeinvaders.org/ | d.gogoboard.org/invaders | Another classic. Move left/right and shoot to defend against alien waves. 3 keys. |
| Breakout | https://www.coolmathgames.com/0-atari-breakout | d.gogoboard.org/breakout | Bounce the ball to break bricks. Move paddle left/right. 2 keys. |
| Donkey Kong | https://freekong.org/ | d.gogoboard.org/kong | Classic arcade platformer. Climb ladders and jump over barrels to rescue the princess. 3 keys. |
